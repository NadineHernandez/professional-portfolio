Featured Projects

Covid Shopping Risk
The Covid Shopping Risk project is a Web App designed to assist shoppers in building their schedule around their own health and safety. The idea is that the user would search for a shopping location and be provided a time table estimating risk of exposure based on store population density throughout the day from Google analytics and the Covid-19 data for that area.
This project was created for the 2020 IBM Call for Code to address the Covid-19 pandemic. It was developed by myself and my team from Cognizant and ranked in our company's Top 20 Submissions. A significant portion of the functionality of this project was integrated into and built upon in successive projects which were presented to clients.
You can find this project on github.
St. Andrew UMC Preschool
This project is a redesign concept for St. Andrew UMC Preschool's website. My focus was to modernize the UI while retaining the existing content.
